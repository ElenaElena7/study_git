git init. Инициализирует новый репозиторий Git и начинает отслеживание существующего каталога. git clone. Создаёт локальную копию проекта, который уже существует удалённо. Клон включает в себя все файлы проекта, журнал и ветви. git add. Подготавливает изменение. Git отслеживает изменения в базе кода разработчика, но для включения изменений в журнал проекта необходимо подготавливать их и создавать моментальные снимки.
